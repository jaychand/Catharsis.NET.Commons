static string Catharsis.Commons.ReflectionAttributesExtensions.Description ( this MemberInfo  self)
static

Returns a value of either DescriptionAttribute, DisplayAttribute or DisplayNameAttribute (whatever is present and found first) for a given class member.

Parameters
selfMember of the class or Type itself.
Returns
Description for a given class self . If self has a DescriptionAttribute, its value is returned. If it has a DisplayAttribute, its description property is returned. If it has a DisplayNameAttribute, its display name property is returned. If there is neither of these attributes on a self , a null is returned.
Exceptions
ArgumentNullExceptionIf self is a null reference.
