static Delegate Catharsis.Commons.ReflectionExtensions.Not ( this Delegate  self,
Delegate  other 
)
static

Removes the last occurrence of the invocation list of a delegate from the invocation list of another delegate.

Parameters
selfThe delegate from which to remove the invocation list of other .
otherThe delegate that supplies the invocation list to remove from the invocation list of self .
Returns
A new delegate with an invocation list formed by taking the invocation list of self and removing the last occurrence of the invocation list of other , if the invocation list of other is found within the invocation list of self . Returns self if other is null or if the invocation list of other is not found within the invocation list of self . Returns a null reference if the invocation list of other is equal to the invocation list of self or if self is a null reference.
Exceptions
ArgumentNullExceptionIf self is a null reference.
See also
Delegate.Remove(Delegate, Delegate)
