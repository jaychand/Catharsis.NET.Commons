static bool Catharsis.Commons.ConvertExtensions.Boolean ( this Convert  self,
object  subject 
)
static

Converts target object to the bool value, using non-strict approach.

The following algorithm is used to determine how to perform such conversion:

  • If subject is a null reference, the result is false.
  • If subject is a bool value, it's returned as it is.
  • If subject is a positive numeric value, the result is true, if it's negative or zero - the result is false.
  • If subject is a string, true is returned if it contains at least one character, false otherwise.
  • If subject implements IEnumerable, true is returned if it contains at least one element, false otherwise.
  • If subject is a FileInfo value, true is returned if associated file is not empty, false otherwise.
  • If subject is a Stream value, true is returned if it's not empty, false otherwise.
  • If subject is a Match value, true is returned if it's successful, false otherwise.
