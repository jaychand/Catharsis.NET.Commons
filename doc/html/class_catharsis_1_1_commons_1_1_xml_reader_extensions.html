Translates XML content from specified XmlReader into a dictionary.

Attributes in XML document are translated to string keys and values, nodes become dictionaries with string keys themselves.

Parameters
selfXmlReader used for reading of XML data which is to be converted to IDictionary{string, object} instance.
closeWhether to automatically close a self after conversion of XML data is finished.
Returns
Dictionary that follows the structure of XML data from XmlReader.
